# Phase 6: Acceleration Comparison Mode - Context

**Gathered:** 2026-04-10
**Status:** Ready for planning
**Source:** discuss-phase

## Phase Boundary

Добавить режим сравнения для нескольких попыток ускорения. Фаза включает:
- Создание компонента AccelerationComparison
- Реализацию выбора попыток для сравнения
- Добавление сравнения кривых ускорения (наложенные кривые)
- Добавление таблицы метрик сравнения с дельта-метриками
- Реализацию подсветки сравнения
- Добавление переключателя режима сравнения
- Добавление фильтрации сравнения (лучшие/худшие попытки по времени)
- Экспорт функциональности (не требуется в этой фазе)

## Implementation Decisions

### Размещение режима сравнения
- **Отдельная вкладка "Сравнение"** рядом с вкладкой "Ускорение"
- **Причина:** Следует паттерну из Фазы 3 (отдельные вкладки для разных режимов визуализации)

### Метод выбора попыток
- **Чекбоксы рядом с каждой попыткой в таблице AccelerationTable**
- **Мульти-селект:** пользователь может выбрать несколько попыток
- **Причина:** Интуитивный паттерн для выбора нескольких элементов

### Визуализация сравнения
- **Наложенные кривые с разными цветами** (линейные графики)
- **Каждая попытка:** отдельный dataset с уникальным цветом
- **Причина:** Эффективно для сравнения формы кривых, соответствует существующему паттерну в AccelerationTab

### Таблица метрик сравнения
- **Выбираемые колонки** (как в AccelerationTable) + **дельта-метрики**
- **Дельта-метрики:** разница относительно лучшей попытки по времени, дистанции, мощностям
- **Ключевые метрики:** время, дистанция, пиковая мощность, средняя мощность, падение батареи
- **Причина:** Пользователь выбирает что показывать, дельта показывает разницу с лучшей попыткой

### Фильтрация сравнения
- **Фильтрация по времени:** лучшие/худшие попытки по времени ускорения
- **Причина:** Время - основной показатель качества ускорения

### Экспорт
- **Не требуется в этой фазе**
- **Причина:** Может быть добавлено в будущей фазе при необходимости

### Claude's Discretion
- Детали реализации компонента AccelerationComparison
- Как определять "лучшую попытку" для дельта-метрик (минимальное время или другой критерий)
- Цветовая палитра для наложенных кривых (использовать существующую ATTEMPT_COLORS)
- UI для переключателя между вкладками "Ускорение" и "Сравнение"
- Интеграция с существующим useAccelerationState hook
