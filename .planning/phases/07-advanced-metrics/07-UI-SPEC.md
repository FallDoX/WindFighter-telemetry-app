## Component-Specific UI Contracts

### AccelerationComparison — Power Curve Toggle (Plan 7.4)

**Placement:** Header section, next to existing filter buttons

**Toggle Button:**
- Base style: `px-3 py-1.5 rounded-lg text-xs font-semibold transition-all border`
- Inactive state: `bg-slate-700/50 border-slate-600 text-slate-400 hover:bg-slate-600/70`
- Active state: `bg-green-500/20 border-green-500/50 text-green-300`
- Label: "Мощность"

**Chart Configuration:**
- Dual-axis: Speed on left Y-axis (0-120 km/h), Power on right Y-axis (0-3000W)
- Power curve style: Dashed line (`borderDash: [5, 5]`), same color palette as speed curves
- Right Y-axis title: "Мощность (Вт)"
- Right Y-axis grid: Disabled (`drawOnChartArea: false`)

### AccelerationTable — Sorting (Plan 7.6)

**Header Interaction:**
- Cursor: `cursor-pointer hover:text-slate-200 transition-colors`
- Sort indicator: Small arrow (↑/↓) next to column label
- Active sort column: No visual distinction beyond indicator
- Click behavior: Toggle direction if same column, set column + asc if different

**Sort Indicator Style:**
- Size: `text-xs`
- Color: Inherits from parent (text-slate-300)

### AccelerationTable — Filtering (Plan 7.7)

**Filter Section:**
- Container: `mb-4 p-4 bg-white/5 rounded-xl border border-white/10`
- Header: Flex row with title and reset button
- Title: "Фильтры" (text-sm font-semibold text-slate-300)
- Reset button: `text-xs text-slate-400 hover:text-slate-200 transition-colors`

**Filter Inputs:**
- Grid: 2 columns with gap-4
- Label: `text-xs text-slate-400 block mb-1`
- Input: `w-full px-3 py-1.5 rounded-lg bg-white/5 border border-white/10 text-slate-200 text-sm focus:outline-none focus:ring-2 focus:ring-blue-500`
- Placeholder: "Нет" (None) when empty

### AccelerationTable — Threshold Warnings (Plan 7.8)

**Visual Warning Styles:**
- High power (peakPower > powerThreshold): `text-red-400 font-semibold`
- High temperature (averageTemperature > temperatureThreshold): `text-orange-400 font-semibold`
- Applied to TableCell only, not entire row

### AccelerationConfig — Metrics Thresholds (Plan 7.8)

**Threshold Section:**
- Placement: Below presets, separated by `mt-4 pt-4 border-t border-white/10`
- Header: "Пороги метрик" (text-xs text-slate-400 mb-2)
- Grid: 2 columns with gap-2

**Threshold Inputs:**
- Label: `text-xs text-slate-400 block mb-1`
- Input: `w-full h-8 text-xs` (uses Input component)
- Default values: 2500W for power, 45°C for temperature
- Placeholder: Shows default value when empty

## Glassmorphism Consistency

All Phase 7 UI elements follow established glassmorphism pattern:
- Backgrounds: `bg-slate-800/50` or `bg-white/5`
