# Руководство по управлению графиком

> Данное руководство описывает все способы взаимодействия с графиком телеметрии для анализа данных поездки.

## Общее управление

График телеметрии поддерживает несколько способов взаимодействия для анализа данных поездки.

---

## Навигация по времени

### Перемещение (Pan)

**Как:** Зажать левую кнопку мыши и двигать влево/вправо

**Что делает:** Сдвигает видимый диапазон времени без изменения масштаба

**Ограничения:** Нельзя выйти за границы данных поездки

---

## Масштабирование (Zoom)

### Колесико мыши с Shift

**Как:** Зажать `Shift` + крутить колесико мыши

- **Вверх (от себя):** Уменьшить масштаб (видеть больше)
- **Вниз (к себе):** Увеличить масштаб (детализация)

**Особенность:** Zoom происходит относительно позиции курсора

### Двойной клик

**Как:** Двойной клик по графику

**Что делает:** Приближает к месту клика (2x)

### Кнопки управления

Над графиком расположены кнопки:

| Кнопка | Действие |
|--------|----------|
| **+** | Увеличить масштаб (приблизить) |
| **Сброс** | Вернуться к полному виду поездки |
| **-** | Уменьшить масштаб (отдалить) |

---

## Инфобар (Info Bar)

### Что показывает

При наведении на график появляется панель с текущими значениями:
- **Время** — точный timestamp
- **Скорость** — км/ч
- **Мощность** — Вт
- **Ток** — А
- **Напряжение** — В
- **PWM** — %

### Фиксация данных

**Как:** Кликнуть по графику

**Что происходит:**
- Данные в инфобаре "замораживаются"
- Пояляется значок LOCK
- Курсор остаётся на выбранной точке

**Как разморозить:** Кликнуть снова или нажать кнопку "Фикс"

---

## Привязка курсора (Snap Mode)

### Включение

**Кнопка:** "Привязка"

**Когда включено:**
- Курсор "прилипает" к ближайшей точке данных
- Точное чтение значений (не интерполяция)

**Когда выключено:**
- Свободное перемещение курсора
- Плавное отображение значений

---

## Типы графиков

### Линейный (Линия)

- Показывает изменение параметров во времени
- Непрерывные линии между точками
- Лучше для анализа трендов

### Точечный (Точки)

- Показывает зависимости между параметрами
- Каждая точка — одно измерение
- Полезно для поиска корреляций (например, скорость vs мощность)

**Как переключить:** Кнопки "Линия" / "Точки" над графиком

---

## Фильтрация данных

### Скрыть простои

**Кнопка:** "Скрыть простои" / "Без простоев"

**Что делает:** Убирает из графика периоды, когда:
- Скорость < 5 км/ч
- Длительность > 30 секунд

**Зачем:** Убирает стоянки на светофорах, остановки

### Фильтр данных

**Кнопка:** "Фильтр" + иконка настроек

**Параметры:**

| Параметр | Описание | Диапазон |
|----------|----------|----------|
| **Разрыв времени** | Максимальный допустимый разрыв между точками | 1-60 сек |
| **GPS лимит** | Максимальная скорость GPS (убирает телепортации) | 0-500 км/ч |
| **Лимит колеса** | Максимальная скорость колеса | 0-300 км/ч |

---

## Управление данными

### Toggle Chips (кнопки под графиком)

Кнопки для включения/выключения отображения параметров:

**Скорость:**
- Скорость (основная)
- GPS скорость

**Мощность:**
- Мощность
- Ток
- Фазный ток

**Система:**
- Напряжение
- Заряд батареи (%)
- Температура
- Torque
- PWM

**Как использовать:** Кликнуть для вкл/выкл. Активные кнопки подсвечены.

---

## Таймлайн (Timeline)

### Нижняя панель

Показывает полную длительность поездки с выделенным текущим диапазоном.

### Управление

- **Drag по центру:** Перемещение окна просмотра
- **Drag краёв:** Изменение масштаба (растягивание/сжатие)
- **Click:** Переместить окно на выбранное место

---

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Shift` + колесо | Zoom |
| `Двойной клик` | Zoom in 2x |
| `Клик` | Фиксация инфобара |

---

## Ограничения

- **Минимальный zoom:** 1 секунда (нельзя зумить ближе)
- **Максимальный zoom:** Вся поездка (нельзя отдалиться дальше)
- **Границы данных:** Нельзя выйти за пределы поездки

---

## Советы по использованию

1. **Детальный анализ:** Shift+колесо на нужном участке
2. **Сравнение участков:** Фиксация (клик) для сохранения значений
3. **Поиск аномалий:** Фильтр данных + точечный график
4. **Оценка ускорения:** Скрыть простои для чистоты графика
